# NUTRIYA - MVP

Proyecto académico de innovación tecnológica basado en inteligencia artificial.

**NUTRIYA** es una aplicación móvil inteligente para registro visual de comidas, recordatorios adaptativos y acompañamiento motivacional para la mejora de hábitos alimentarios.

## Estructura del repositorio

```text
NUTRIYA-MVP/
├── mobile_app/          # Propuesta de aplicación móvil
├── backend_api/         # Servicios API para autenticación, comidas, IA y recordatorios
├── database/            # Modelo lógico, script base y diagramas
├── docs/                # Documentación académica y planificación
├── prototypes/          # Interfaces UI y wireframes
├── assets/              # Identidad visual del proyecto
└── tests/               # Evidencias o pruebas iniciales
```

## Módulos principales

- Registro e inicio de sesión.
- Perfil nutricional.
- Registro de comida por foto.
- Registro de comida por texto.
- Análisis nutricional con IA.
- Dashboard diario.
- Recordatorios y frases motivacionales.
- Panel administrativo.

## Nota académica

Este repositorio corresponde a la preparación del Producto Académico N.° 2. Incluye estructura inicial, documentación, prototipos y diseño lógico de base de datos para evidenciar la construcción y preparación del entorno de trabajo.
